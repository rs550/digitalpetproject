About: Meet Blueberry, a digital pet ghost for your browser! A simple three button digital pet with idle animations.
      Simple vanilla JavaScript, HTML, CSS. 

Project Status: This is a practice project to remind myself how to use HTML/CSS and practice basic JavaScript to build off.
      Blueberry may appear again in future projects but the goal for this project was to learn and practice.

How to Use: Copy the repo and open main.html in your browser! Once loaded if no button is selected, no change will happen.
      Once a button is pressed the program will independently decrease status bars, when the top (health) bar is empty
      the program resets. Don't let Blueberry die again!
